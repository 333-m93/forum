# Le Dojo

Forum communautaire dedie aux sports de combat.

## Stack

- Go 1.25
- PostgreSQL
- Docker

## Fonctionnalites

- Inscription / Connexion / Deconnexion
- Chat par categorie (MMA, Boxe, Muay Thai, JJB, Grappling, etc.)
- Reactions emoji sur les messages
- Profils utilisateurs (avatar, bio, mot de passe)
- Panel admin (dashboard, ban/unban, suppression de messages)
- Mode sombre / clair

## Variables d'environnement

| Variable | Description |
|----------|-------------|
| `DB_HOST` | Hote PostgreSQL |
| `DB_USER` | Utilisateur PostgreSQL |
| `DB_PASS` | Mot de passe PostgreSQL |
| `DB_NAME` | Nom de la base |
| `APP_PORT` | Port du serveur (defaut : 8080) |
